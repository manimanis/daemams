Data Analysis

    Data - Information - Knowledge - Wisdom

    Definitions

    Data :

    Unstructured collection of facts and figures which relay something specific, but which are not organized in any way.

    Data is a set of distant facts about events.

    Information :

    • contextualized, categorized, calculated and condensed data.
    • data with relevance and purpose.
    • convey a trend in the environment, or a pattern in sales.

    Information comes from giving context to data.

    Knowledge :

    Framed experience, values, contextual information, expert insight and grounded intuition.

    Knowledge is created when an individual's judgments, insights, ideas, and experiences are added to the results of analyzing the data and information. Knowing how to use information for a purpose is knowledge.
