    Data Types

    Identifying data types is important, as it allows us to understand the types of analyses that we can perform and the plots that we can build.

    There are mainly two types of data:

    • quantitative data, like the number of dogs, they takes numeric values that allow us to perform mathematical operations.
    • and categorical data, like the breed, frequently used to label a group or set of items.

    Quantitative data

    Quantitative data types can be divided into:

    • continuous, can take on any numeric value including decimal values, like for example the age of a dog - we can measure the units of the age in years, months, days, hours, seconds, but there are still smaller units that could be associated with the age.
    • and discrete, takes on countable values, like for example the number of dogs

    Categorical data

    Further, categorical data types can be divided into:

    • ordinal, ranked values, for example rating from very positive to very negative,
    • and nominal, do not have a rank order, for example breeds: Lab, Poodle, Pug, Chihuahua, etc.
